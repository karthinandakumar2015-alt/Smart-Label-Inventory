# Smart Label Inventory

A premium inventory management prototype for archaeological artifacts with smart tracking features.

## Features
- **Modern Dashboard**: elegant glassmorphism design with AVENIR and Poppins typography.
- **Role-Based Access Control**:
  - **Admin**: Add/Delete artifacts, View details.
  - **Viewer**: View details only.
- **Smart Artifacts**:
  - **Location Tracking**: Interactive map integration.
  - **QR Code**: Auto-generated QR codes.
  - **Battery Health**: Tracker battery monitoring.
- **Audit Trail**: Tracks who added, viewed, or deleted artifacts.

## How to Run

1. Simply open `index.html` in your web browser (Chrome, Edge, Firefox).
2. No server installation required.

## Login Credentials

- **Admin Access**:
  - Username: `admin` (or any name)
  - Role: Select **Admin** from dropdown.

- **Viewer Access**:
  - Username: `guest` (or any name)
  - Role: Select **Viewer** from dropdown.
